What is ARR?

ARR is a full-stack web platform where researchers upload, discover,
and cite academic papers alongside their source code. Every paper
gets an auto-generated citation, a download counter, and category tags.

Core Modules

Upload Form

Drag-and-drop PDF + ZIP upload with metadata fields and progress indicator.

Citation Generator

Auto-builds APA / MLA / IEEE citations from paper metadata with one-click copy.

Search & Filter

Full-text search by title, author, category tags, and date range filters.

Download Tracker

Logs every download with timestamps; live counter on each paper card.

PDF Preview

In-browser PDF viewer with page navigation before download is required.

Admin Panel

Manage papers, users, tags, and moderation queue via a secure dashboard.

User Profile

Personal dashboard showing uploads, saved papers, and citation history.

Notifications

Real-time alerts for paper approvals, new citations, and download milestones.

Category Tags

Hierarchical taxonomy for CS, ML, Physics, Bio, and 30+ custom domains.

REST API + Postman

Full CRUD endpoints tested via Postman collection; docs available for all routes.
